常见问题

FAQ

快速解答您的疑问

Compiler IDE 支持哪些操作系统？
目前 Compiler IDE 支持 Windows 10/11 64位系统。未来计划支持 macOS 和 Linux。
安装包为什么有不同大小？
提供两个版本：精简版（约400MB）不含编译器，需自行安装MinGW；完整版（约1GB）内置最新版 TDM-GCC 编译器，开箱即用。
如何配置编译器路径？
完整版自动配置，无需手动设置。精简版需在设置中手动指定 MinGW 的 bin 目录路径，例如：C:\mingw64\bin
竞赛模式有什么特别之处？
竞赛模式专为算法竞赛设计，提供倒计时/正计时器，可禁用自动补全、自动保存等功能，让你专注于编码。还支持快速编译运行快捷键。
如何获取更新？
IDE 启动时会自动检查更新。如有新版本，会弹出提示，点击即可一键更新。也可在"帮助"菜单中手动检查更新。
Compiler IDE 与 Dev-C++ 有什么区别？
Compiler IDE 拥有更现代化的界面、更好的C++20/23支持、集成终端、竞赛模式、系统性能监控等功能。Dev-C++ 自2015年后已停止更新，而 Compiler IDE 持续维护更新中。
